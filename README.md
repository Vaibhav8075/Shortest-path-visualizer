# Shortest Path Visualizer

An interactive visualization tool for pathfinding algorithms including Dijkstra's and A*.

## Features
- Visual grid-based pathfinding
- Multiple algorithm support (Dijkstra, A*)
- Side-by-side algorithm comparison
- Maze generation
- Adjustable speed controls

## Files
- `app.js` - Main application logic
- `index.html` - UI structure
- `style.css` - Styling
- `algorithms/` - Algorithm implementations
  - `dijkstra.js` - Dijkstra's algorithm
  - `astar.js` - A* algorithm
- `utils/` - Utility functions
  - `priorityQueue.js` - Priority queue implementation

## Usage
Open `index.html` in a web browser or serve with a local server.
